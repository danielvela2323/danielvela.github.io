App Android – Comercio Directo

Aplicación Android enfocada en eliminar intermediarios mediante la conexión directa entre productores y empresas.

Android Java SQLite MySQL
Ver código Demo / APK PDF

El problema

Los pequeños productores suelen vender sus productos a intermediarios que reducen significativamente su margen de ganancia.

La solución

Se desarrolló una aplicación móvil que permite la conexión directa entre productores y compradores, eliminando intermediarios.

Funcionalidades principales

  • Registro y login de usuarios
  • Validación de correo electrónico
  • Diseño enfocado en usabilidad

Lo que aprendí

  • Diseño de flujos de usuario en apps móviles
  • Conexión con bases de datos
  • Mejores prácticas de organización de código
